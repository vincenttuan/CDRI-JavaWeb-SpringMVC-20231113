Hello SpringMVC

    HelloController

  1. 取得 Welcome SpringMVC 字串
    hello/welcome
  2. ? 帶參數 限制: name 參數是一定要有的(預設), age 參數是不一定要有的, 初始值 = 0
    hello/hi?name=John&age=18
    hello/hi?name=Mary&age=20
    hello/hi?name=Helen
  3. BMI Lab 練習 (限制: h, w 是一定要有)
    hello/bmi?h=170.0&w=60.0
  4. 同名多筆的資料參數 計算平均年齡
    hello/age?age=17&age=21&age=20
  5. Lab 練習: 得到多筆 score 資料, 印出: 最高分=?、最低分=?、平均=?、總分=?、及格分數=?、不及格=?
    hello/exam?score=80&score=100&score=50&score=70&score=30
  6. 多筆參數資料轉 Map
    hello/personMap?name=John&age=18&score=80&pass=true
  7. 多筆參數資料轉 bean (例如: person)
    hello/person?name=John&age=18&score=80&pass=true
  8. 路徑參數 @PathVairable
    hello/java_score/75?name=John
    hello/java_score/45?name=Mary
